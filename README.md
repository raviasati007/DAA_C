# 📘 Design & Analysis of Algorithms – Practical List (C)

This repository contains **C language implementations** for the **DAA Practical List** (Session 2026–27) at *S. B. Jain Institute of Technology, Management & Research, Nagpur*.  

Each practical is inspired by real-world scenarios and mapped to course outcomes.

---

## 🧪 Practical List

| Practical | Problem Statement | Algorithm/Concept |
|-----------|------------------|-------------------|
| **[Pre-Lab](ca://s?q=Explain_pre_lab_prime_number_analysis)** | Analyze time & space complexity of prime number generation up to `n` | Complexity Analysis |
| **[Practical 1](ca://s?q=Explain_practical_1_quick_sort_logs)** | Sort massive server logs chronologically using timestamps | Quick Sort |
| **[Practical 2](ca://s?q=Explain_practical_2_binary_search_archive)** | Locate first occurrence of manuscript Archive ID efficiently | Binary Search |
| **[Practical 3](ca://s?q=Explain_practical_3_strassen_matrix_multiplication)** | Optimize spacecraft trajectory with matrix multiplication | Strassen’s Algorithm |
| **[Practical 4](ca://s?q=Explain_practical_4_kruskal_mst)** | Minimize fiber internet network installation cost | Kruskal’s MST |
| **[Practical 5](ca://s?q=Explain_practical_5_dijkstra_ambulance_routing)** | Real-time ambulance routing in Mumbai | Dijkstra’s Algorithm |
| **[Practical 6](ca://s?q=Explain_practical_6_tsp_dynamic_programming)** | Optimize drone delivery routes | Travelling Salesman Problem (Dynamic Programming) |
| **[Practical 7](ca://s?q=Explain_practical_7_longest_common_subsequence)** | Compare DNA sequences for evolutionary traits | Longest Common Subsequence |
| **[Practical 8](ca://s?q=Explain_practical_8_bellman_ford_arbitrage)** | Detect arbitrage opportunities in currency exchange | Bellman-Ford Algorithm |
| **[Practical 9](ca://s?q=Explain_practical_9_n_queens_backtracking)** | Place antennas without interference on a grid | N-Queens Problem |
| **[Practical 10](ca://s?q=Explain_practical_10_depth_first_search_social_graph)** | Trace social media post reach | Depth-First Search |
| **[Post-Lab](ca://s?q=Explain_post_lab_open_ended_practical)** | Open-ended practical | Student-defined |
